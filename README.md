# Viva Buddy AI 🎓

Viva Buddy AI is an AI-powered quiz application designed to help students prepare for vivas and oral examinations.

Users enter a topic, and the application generates beginner-friendly viva questions using Gemini AI. The quiz includes hints, answer checking, scoring, timers, and detailed performance analysis to help identify areas for improvement.

## Features

* 🤖 AI-generated viva questions using Gemini AI
* 💡 Hint system with reduced scoring
* ⏱️ Question timer
* 📊 Automatic score calculation
* 📝 Detailed answer analysis
* 🌙 Modern dark-themed UI with glow effects
* 🔄 Dynamic question generation for any topic

## Tech Stack

* HTML
* CSS
* JavaScript
* Gemini API

## How It Works

1. Enter a topic (e.g., Python, Java, Arrays).
2. Gemini AI generates viva-style questions.
3. Answer each question within the time limit.
4. Use hints if needed.
5. View your final score and detailed answer analysis.
